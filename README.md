# USSD MVOLA
This project simulates a USSD **MVola** menu in Java, running in the terminal.  
It lets you navigate through a main menu and sub-menus just like on a telephone.

---

## Start the app
1. **Compiling files :**
```
javac Main.java MenuHandler.java SubMenu.java
```

2. **Executing files :**
```
java Main
```

## Use Linter
```
./lint.sh
```
